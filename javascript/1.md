# Append item to array

- Using `array.push()`

   ```js
   array.push("ğŸ’»")
   ```

- Using `array.splice`

   ```js
   array.splice(array.length, 0, "ğŸ’»")
   ```

> console.log(original) // ["ğŸ‘¦", "ğŸ¥ª", "ğŸ’»"]

**Note:** These above arrays mutate the original array.

- Using spread operator `...`

   ```js
   let new_array = [...original, "ğŸ’»"]
   ```

- Using `array.concatenate()`

   ```js
   let new_arry =  original.concatenate("ğŸ’»")
   ```

> console.log(original) // ["ğŸ‘¦", "ğŸ¥ª"]

> console.log(new_array) // ["ğŸ‘¦", "ğŸ¥ª", "ğŸ’»"]

**Note:** Above two method don't mutate original array, but always create new array
